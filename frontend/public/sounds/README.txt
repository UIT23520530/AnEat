Notification sound file placeholder.

To add a real notification sound:
1. Find a notification.mp3 file (bell, ding, or chime sound)
2. Place it in this directory at: /sounds/notification.mp3
3. Recommended: Short sound (1-2 seconds), ~100-200KB

Free sound sources:
- https://mixkit.co/free-sound-effects/notification/
- https://freesound.org/
- https://pixabay.com/sound-effects/

For now, the app will try to play this file but will gracefully fail if not found.
